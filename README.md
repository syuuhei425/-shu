# 📊 家計簿アプリ

シンプルで使いやすい家計簿PWAアプリです。

## 🚀 今すぐ使う

### ウェブ版で開く
https://syuuhei425.github.io/-shu/kakeibo4.html

## 📱 スマートフォンのホーム画面に追加

### iPhone（Safari）
1. 上記URLをSafariで開く
2. 下部の共有ボタン（矢印マーク）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

### Android（Chrome）
1. 上記URLをChromeで開く
2. 右上のメニュー（⋮）をタップ
3. 「アプリをインストール」を選択
4. 「インストール」をタップ

## ✨ 主な機能

- 💰 収入・支出の記録
- 📌 固定費・固定収入の登録（毎月自動反映）
- 📊 カテゴリ別の支出内訳グラフ
- 📈 月別収支推移の折れ線グラフ
- 🗓️ カレンダーから日付選択
- ✏️ メモ機能
- 🔒 全データはデバイス内に保存（サーバー送信なし）
- 🌐 オフライン対応

## 💡 使い方のコツ

- **固定費を登録すると**: 毎月の明細に自動的に反映されます
- **カテゴリを自由に追加**: 「その他」から カスタム項目を入力できます
- **グラフを見るには**: 「グラフ」タブから確認できます
- **データのバックアップ**: ブラウザのキャッシュクリアでデータが消えます（注意）

## 🔧 技術

- HTML5 + CSS3 + JavaScript
- PWA（Progressive Web App）
- Service Worker対応
- Chart.js（グラフ表示）
- オフライン動作対応
